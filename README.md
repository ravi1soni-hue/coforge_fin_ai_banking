# Coforge Fin AI

This project powers financial orchestration and decisioning agents for corporate treasury and banking use cases.

## Features
- Agent orchestration for finance workflows
- Data ingestion and vector search
- Scalable supplier and user data management
- API adapters for flexible queries

## Getting Started
1. Clone the repo
2. Install dependencies: `npm install`
3. Run the server: `npm start`

## Deployment
This project is deployed via Railway. Push to the main or deployment branch to trigger redeploy.

## Contributing
- Create feature branches for new work
- Use clear commit messages
- Keep code stable and tested
